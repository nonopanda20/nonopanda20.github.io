Methodology

How this navigator was built

01

Data Collection

Article metadata harvested from OpenAlex API — a free, open academic database covering 200M+ works.

publication_year:2005-2026, type:article|review
02

Journal Selection

74 journals across business and economics, including UTD 24 and top 50 economics journals by impact factor.

24 Business · 50 Economics · 7 disciplines
03

Data Processing

Abstracts reconstructed from OpenAlex inverted index. Text cleaned and non-research articles filtered out.

96,176 valid articles after cleaning
04

Data Compression

JSON compressed using short field keys. Result: 58 MB for 96k articles with full abstracts.

~600 bytes per article average
05

Frontend Architecture

Single-page application with inline CSS + JS. No framework dependencies.

84 KB HTML · 0 external JS · 0 external CSS
06

Auto-Update

Incremental update script fetches new articles monthly. Deduplication by DOI ensures no duplicates.

python3 scripts/update_data.py
