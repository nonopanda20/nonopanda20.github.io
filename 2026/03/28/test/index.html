Basic Properties of Matrix

Definition: Special kinds of matrices

  • Square matrix: $m = n$
  • Diagonal matrix: $m = n$ and $\alpha_{ij} = 0$ for $i \neq j$
  • Identity matrix $I$: diagonal matrix with $\alpha_{ii} = 1$ for all $i$
  • Upper-triangular: $\alpha_{ij} = 0$ for $i > j$
  • Lower-triangular: $\alpha_{ij} = 0$ for $i < j$
  • Symmetric: $\alpha{ij} = \alpha{ji}$ for all $i, j$
  • Idempotent: A square matrix $A$ is called an idempotent matrix if $AA = A$

Definition: 转置和逆

Transpose: The $n \times m$ matrix $A’$ is the transpose of the $m \times n$ matrix $A$ if $a’{ji} = a{ij}$.

Properties:

  1. $(A’)’ = A$
  2. $(A + B)’ = A’ + B’$
  3. $(AB)’ = B’A’$

Inverse: The inverse of matrix $A$, $A^{-1}$, is defined only if $A$ is a square matrix, and $AA^{-1} = A^{-1}A = I$.

A square matrix may or may not have an inverse.
If a square matrix does have an inverse, it is called a nonsingular matrix; otherwise, it is called a singular matrix.

Properties:

  1. If an inverse exists, it is unique.
  2. $( A^{-1} )^{-1} = A$
  3. $( AB )^{-1} = B^{-1} A^{-1}$
  4. $( A’ )^{-1} = ( A^{-1} )’$

Rank and Determinant

Definition: Elementary row operations

Elementary row operations:

  1. Interchange two rows of a matrix
  2. Change a row by adding to it a multiple of another row
  3. Multiply each element in a row by the same nonzero number

The elementary row operations can be performed on a matrix $A$ by premultiplying $A$ by certain special matrices called elementary matrices. The matrices $E{ij}$, $E{ij}(r)$, and $E_i(r)$, which are obtained by performing elementary row operations on the identity matrix, are called elementary matrices.

Definition: Row echelon form

A row of a matrix has $k$ leading zeros if the first $k$ elements of the row are all zeros and the $(k+1)^{\text{th}}$ element of the row is not zero. A matrix is in row echelon form if each row has more leading zeros than the row preceding it.

Definition: Pivot and Rank

A pivot of a matrix in row echelon form is an element which is the first nonzero element in its row.

A row echelon matrix in which each pivot is a 1 and in which each column containing a pivot contains no other nonzero entries is said to be in reduced row echelon form.

The rank of a matrix $A$, denoted $\operatorname{rank}(A)$ or $r(A)$, is the number of nonzero rows in any of its row echelon forms (equivalent to the number of pivots).
