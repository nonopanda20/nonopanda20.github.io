Methodology

How this navigator was built

01

Data Collection

Article metadata harvested from OpenAlex API — a free, open academic database covering 200M+ works.

GET /works?filter=...publication_year:2005-2026
02

Journal Selection

74 journals across business and economics, including UTD 24 and top 50 economics journals.

24 Business · 50 Economics
03

Data Processing

Abstracts reconstructed from OpenAlex inverted index. Text cleaned and non-research articles filtered.

96,176 valid articles
04

Data Compression

JSON compressed using short field keys. Result: 58 MB for 96k articles with full abstracts.

~600 bytes per article
05

Frontend Architecture

Single-page app with inline CSS + JS. No framework dependencies.

84 KB HTML · 0 external JS
06

Auto-Update

Incremental update fetches new articles monthly. Deduplication by DOI.

python3 scripts/update_data.py
