Methodology

How this navigator was built

01

Data Collection

Article metadata harvested from OpenAlex API — a free, open academic database covering 200M+ works. Fetched using cursor-based pagination (200 per page) with polite rate limiting.

GET /works?filter=primary_location.source.issn:{ISSN},publication_year:2005-2026,type:article|review
02

Journal Selection

74 journals across business and economics, including UTD 24, top 50 economics journals by impact factor, and key field journals in labor, development, health, and environment.

24 Business · 50 Economics · 7 disciplines
03

Data Processing

Abstracts reconstructed from OpenAlex inverted index format. Text cleaned: HTML tags (<scp>, <i>), entities (&amp;), and formula placeholders removed. Non-research articles filtered out.

96,176 valid articles after cleaning
04

Data Compression

Original JSON compressed using short field keys (t/a/ab/y/j/c/d/k) and concatenated without indentation. Result: 58 MB for 96k articles with full abstracts.

~600 bytes per article average
05

Frontend Architecture

Single-page application with inline CSS + JavaScript. No framework dependencies. Data loaded via fetch() API. Charts rendered with pure CSS (no chart libraries). Fonts from Google Fonts CDN.

84 KB HTML · 0 external JS · 0 external CSS
06

Auto-Update

Incremental update script fetches only new articles from the latest year. Run monthly via cron job or manually. Deduplication by DOI ensures no duplicates.

python3 scripts/update_data.py
